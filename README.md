# clibe
use any npm library as a cli

# install
```bash
npm install clibe -g
```
or see usage below for running with `npx`


# usage
```bash
npx clibe <lib-name> [command] [args..]
```
for example   
```bash
clibe 'lib-name' 'some-command' 'args' 'for' 'command'
```

# example
```bash
npx clibe 'gradient-string' 'teen' 'HELLO WORLD'
```
